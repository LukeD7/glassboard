# Glassboard Roadmap

This roadmap is intentionally short.
Glassboard prioritizes polish over features.

---

## v1.0 — Core Clipboard UX

- Stable clipboard capture (text + images)
- Fast, reliable global hotkey
- Visual list UI with keyboard navigation
- Screenshot-first presentation
- Basic pinning
- Menu bar access

---

## v1.1 — Performance & Polish

- Startup and panel open speed improvements
- Better large-clipboard handling
- Accessibility refinements
- UI spacing, animation, and focus polish

---

## v1.2 — Small Experiments (Optional)

- Optional grid layout (behind a flag)
- Improved image handling and previews
- Better visual grouping by time

---

## Explicitly Out of Scope

- Sync or cloud features
- Preferences UI
- Power-user workflows
- Plugin or integration systems
